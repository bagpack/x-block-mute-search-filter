# X Block & Mute Search Filter

X (旧Twitter) の検索結果から、ブロック/ミュートしたアカウントを非表示にするChrome拡張です。

English README: [README.md](README.md)

## 主な機能
- 検索結果ページでブロック/ミュートの投稿を非表示
- ログイン済みセッションからミュート/ブロック一覧を取得
- X上でミュート/ブロック操作をすると即時反映
- ポップアップに一覧件数と非表示アカウント数を表示

## 事前条件
- ブラウザでXにログインしていること

## インストール(ローカル)
1. `chrome://extensions` を開く
2. 右上の「デベロッパーモード」をON
3. 「パッケージ化されていない拡張機能を読み込む」で本フォルダを選択

## 使い方
1. Xの検索ページを開く (`https://x.com/search?...`)
2. 拡張アイコンをクリックしてポップアップを開く
3. 「ミュート/ブロック一覧を取得」をクリック
4. 完了するとポップアップに「取得完了」が表示されます

## ブロック/ミュートを追加した後の反映手順
通常はX上の操作で即時反映されます。反映されない場合は再取得してください。
1. 拡張アイコンをクリック
2. 「ミュート/ブロック一覧を取得」をクリック
3. ポップアップの「取得完了」表示を確認

## 補足
- 動作対象はXの検索ページのみです
- 取得したデータは `chrome.storage.local` に保存されます
- 取得が失敗する場合は `https://x.com/settings/muted/all` と
  `https://x.com/settings/blocked/all` を一度開いてAPI情報を再取得してください
